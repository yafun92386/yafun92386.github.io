[Hexo] Markdown語法整理 + NexT的Tag Plugin

Posted on 2020-11-30 In Web , Hexo

Hexo使用Markdown來編寫文章，因此來整理一些需要的語法，還有補充支援Next的Tag Plugins。

1. Markdown

Markdown是一種標記式語言，減少標籤、格式指令以強調可讀性。

1.1 文字樣式

1) 標題

Stext形式

分為6個層級，於行首插入#與空白。

Atx形式

採用底線形式分為最高階=和第二階-。

2) 分隔線

用3個以上的星號*、減號-、底線_來建立分隔線，行內不能有其他符號，可以插入空白。

3) 文字字體

粗體 **粗體標示** 、 __粗體標示__

斜體 *斜體標示*

粗斜體 ***粗斜體***

刪除線 ~~刪除線~~

底線 <u>底線</u>

4) 引言

使用email形式的引言，於行首加上>，亦可使用階層、標題、清單關係語法。

5) 程式碼

文字程式碼

輸入1個反引號`涵蓋。

段落程式碼

採用3個反引號`(英文半形)涵蓋，第一列後可以指定語言類型。

6) 公式

以$涵蓋表示行內公式，$$涵蓋表示獨立公式。

其餘語法參考:
MathJax basic tutorial and quick reference
LaTeX/Mathematics
如何在 Markdown 輸入數學公式及符號
LATEX語法筆記

1.2 格式樣式

1) 清單

支援有序清單和無序清單，可使用階層式。

有序清單

使用文字接英文句點+空格。

若段落中以數字+.作為開頭，可改為數字+\.

無序清單

使用星號*、加號+、減號-+空格。

2) 待辦事項

[ ] 表示空框框，[x]顯示打勾。

3) 表格

第一列為標題列，第二列以---區隔，以|增加欄位，欄寬會自動分配不用對齊。

第二列可決定對齊方式。
